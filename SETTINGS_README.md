# Settings — Future AI Onboarding Integration

## Note (April 2026)

The **Contact Fields** and **Lead Sources** configuration currently lives in the Settings page
under manual UI controls (add/remove/reorder).

**In a future release**, this configuration will be handled as part of the **AI-guided onboarding flow**:

1. During organization setup, the AI agent will ask the business owner about their sales process
2. Based on the conversation, the agent will automatically suggest and configure:
   - Lead sources relevant to the business vertical
   - Custom contact fields (e.g., credit score for finance, property size for real estate)
   - Pipeline stages
   - Commission structures
3. The owner can review and approve the AI's suggestions before they're applied
4. The Settings page will remain available for manual adjustments after onboarding

This approach aligns with Kove's anti-CRM philosophy: the system configures itself through
conversation rather than requiring the user to navigate settings forms.
